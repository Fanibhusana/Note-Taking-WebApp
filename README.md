## 🚀 Welcome to the 📝 Note Taking Web Application !

Welcome to the Note Taking web application, a modern and efficient tool to manage your notes with ease. This project is built using Java, JSP, Servlet, Hibernate with JPA, MySQL, SQL, HTML, Bootstrap, Maven, and Apache Tomcat.

## ✨ Features

- **User Registration:** Users can register by providing necessary details.
- **User Login:** Registered users can log in using their credentials.
- **Logout:** Users can log out of their accounts.
- **Edit Profile:** Users can update their profile information.
- **Edit Password:** Users can change their password.
- **Add Note:** Users can add new notes.
- **View Note:** Users can view their saved notes.
- **Update Note:** Users can update existing notes.
- **Delete Note:** Users can delete notes they no longer need.

## 🛠 Technologies Used

- **Backend:** Java, JSP, Servlet, Hibernate with JPA, MySQL
- **Frontend:** HTML, Bootstrap
- **Build Tool:** Maven
- **Server:** Apache Tomcat

## 📚 Usage

### 📝 Register
1. Open the web application in your browser.
2. Navigate to the Register page.
3. Fill in the required details and submit the form to create a new account.

### 🔐 Login
1. Open the Login page.
2. Enter your registered email and password.
3. Click on the Login button to access your account.

### 🚪 Logout
- Click on the Logout button to end your session.

### 🖋 Edit Profile
1. Navigate to the Edit Profile page.
2. Update your profile information.
3. Submit the form to save changes.

### 🔑 Edit Password
1. Navigate to the Edit Password page.
2. Enter your current password and the new password.
3. Submit the form to update your password.

### 🆕 Add Note
1. Navigate to the Add Note page.
2. Enter the note details.
3. Submit the form to save the note.

### 👀 View Note
- Navigate to the View Notes page to view all your saved notes.

### 🔄 Update Note
1. On the View Notes page, click the Edit button next to the note you want to update.
2. Update the note details.
3. Submit the form to save changes.

### 🗑 Delete Note
1. On the View Notes page, click the Delete button next to the note you want to remove.
2. Confirm the deletion.
